The Novels Project gathers digitizations of novels published between 1770 and 1915.

About

The Novels Project gathers and commissions digitizations (scans and digital editions) of novels published between 1770 and 1915. Currently the project focuses on collecting a representative sample (i.e., a simple random sample) of novels published in the British Isles between 1800 and 1836.

Thousands of novels published between 1770 and 1915 survive in library collections around the globe. For example, of the 2,903 novels published between 1800 and 1836 in the British Isles, ninety-nine percent survive. Only a bare majority have been scanned.

Contribute

Contributions are welcome. Please open an issue GitHub if you have located a digitization suitable for inclusion.

API

The API endpoint {{ novels_api_addr }} exposes the following resources:

  • /work/ returns JSON of all {{ stats.novels_count }} bibliographic records with metadata on scanned volumes.
  • /text/{SHA1} returns the full text of the volume uniquely identified by its SHA1 hash.

Bulk data access is available and welcomed. The Novels Project database is backed by a git repository. You can clone that repository or download a snapshot.
